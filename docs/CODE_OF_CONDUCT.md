# Cruzr Car Rental System – Code of Conduct

## Our Commitment
As contributors and maintainers of the **Cruzr Car Rental System**, we are committed to creating a welcoming, respectful, and inclusive environment for everyone regardless of age, background, gender, ability, or experience.

## Expected Behavior
- Use kind and respectful language.
- Provide constructive feedback and help others learn.
- Focus on improving the project and community.
- Follow the contribution guidelines and communication channels.

## Unacceptable Behavior
- Any form of discrimination, harassment, or personal attacks.
- Posting or sharing private information without consent.
- Using offensive or inappropriate language.
- Repeatedly disrupting project discussions or pull requests.

## Responsibilities
Project maintainers have the right to remove or edit comments, commits, or issues that violate this Code of Conduct and may temporarily or permanently ban contributors for inappropriate behavior.

## Reporting Violations
If you notice behavior that violates this Code of Conduct, please **report it directly through the blue support button** located at the bottom-right corner of the website.  
This button opens our secure in-site support chat where you can describe the issue privately.  
All reports are reviewed by the maintainers and handled with fairness and confidentiality.

## Scope
This Code of Conduct applies to all project spaces — GitHub issues, pull requests, discussions, and other communication channels related to the project.
